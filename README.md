# 测试wsl环境是否配置正确

**进入目录** 
```bash
cd test_your_env
```

**编译**
```bash
./build.sh
```
生成的二进制在`{workspace}/test_your_env/bin`目录下


**运行**
```bash
./runAll.sh
```



> 如果报错说明环境可能配置不对
